# twitter_spider
A twitter spider

# Require
<strong>tweepy</strong> : pip install tweepy<br>
<strong>request security</strong> : pip install 'requests[security]'

#Usage:
python app.py screen_name
<br>
Example:
```
python app.py google
```
#Return result

````
User: google 
User ID : 20536157
Follower count : 13124207
Friend count : 439
```
